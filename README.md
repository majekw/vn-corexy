# vn-corexy
Very Narrow CoreXY 3D Printer

This is OpenScad source code to generate hardware parts for vn-corexy.

# Requirements
- OpenScad https://www.openscad.org/
- NopSCADlib library https://github.com/nophead/NopSCADlib
